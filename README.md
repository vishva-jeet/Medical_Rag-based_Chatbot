# Medical Question Answering RAG Chatbot Project

This project lets you ask questions from PDF files using LLaMA and LangChain.

## working step by step :
- Reads PDF files
- Breaks text into pieces
- Stores meaning in a database
- Uses LLaMA to answer questions

## How to run this project:

1. Install Python
2. Install requirements
   pip install -r requirements.txt
3. Put PDFs inside data/pdfs
4. Put model inside models
5. Run
   python src/main.py

## Big file is not mentioned here 

Big files are not uploaded to GitHub, link is given in models folder.
